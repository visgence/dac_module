#Power DAC Module
A DAC biphasic +- 12v DAC that can run off any 3.3v or 5v microcontroller By Visgence Inc.
![DAC Module](images/dac_module_small.jpg?raw=true)

## Specs
* DAC IC: MCP4921
* Input V+
 * MAX: 12V
 * Min 3.3V
* Input V+
 * MAX: -12V
 * Min -3.3V
